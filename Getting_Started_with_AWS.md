```
For Linux
$ curl "https://awscli.amazonaws.com/awscli-exe-linux-x86_64-2.0.30.zip" -o "awscliv2.zip"
$ unzip awscliv2.zip
$ sudo ./aws/install
```

```
For Windows
Download and Install https://awscli.amazonaws.com/AWSCLIV2.msi
```
